## Our Commitment

ZTAS.io is committed to ensuring PhotoSifter is accessible to all users, including those with disabilities. We strive to meet WCAG 2.1 Level AA standards.

## Website Accessibility

This website has been designed with accessibility in mind:

### Navigation
- Keyboard-navigable interface
- Skip-to-content links
- Logical heading structure
- Focus indicators on interactive elements

### Visual Design
- Sufficient color contrast (WCAG AA compliant)
- Resizable text without loss of functionality
- No content that flashes more than 3 times per second
- Support for reduced motion preferences

### Content
- Descriptive link text
- Alt text for images
- ARIA labels where appropriate
- Semantic HTML structure

## Desktop Application

PhotoSifter desktop app includes:
- Keyboard navigation support
- Screen reader compatibility
- High contrast UI elements
- Clear, readable fonts
- Logical tab order

## Known Limitations

We're continuously working to improve accessibility. Current known limitations:
- Some complex interactions may require mouse input
- Custom file dialogs rely on OS accessibility features

## Feedback

We welcome feedback on the accessibility of PhotoSifter. If you encounter barriers or have suggestions:

- **Email:** [support@photosifter.com](mailto:support@photosifter.com)
- **GitHub:** [Open an issue](https://github.com/jasonsutter87/photosifter/issues)

We aim to respond to accessibility feedback within 5 business days.

## Ongoing Efforts

We regularly:
- Test with screen readers (VoiceOver, NVDA)
- Audit color contrast
- Review keyboard navigation
- Update based on user feedback

## Standards

This statement was prepared based on:
- Web Content Accessibility Guidelines (WCAG) 2.1
- Section 508 of the Rehabilitation Act
- EN 301 549 (European accessibility standard)
